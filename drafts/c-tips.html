Type Aliases

Favor using over typedef in general!

  • using <alias> = <type> is equivalent to typedef <type> <alias>, but I'm often confused the order of <type> and <alias> in typedef.
using ll = long long; // typedef long long ll;
using vi = vector<int>; // typedef vector<int> vi;

Also, using can be used with template:

template <class T>
using TT = pair<T, T>;
